O HomeManager é um sistema completo para gerenciamento de elevadores que permite:

  • Cadastro e gerenciamento de clientes
  • Controle de endereços múltiplos por cliente
  • Gestão de contratos com datas de entrega
  • Cadastro completo de elevadores
  • Visualização em calendário das entregas
  • Relatórios e análises de vendas
Sistema desenvolvido com Flask, PostgreSQL e Bootstrap 5
